Description
-----------

Zone Achievement Tracker automatically tracks the achievement for 
completing a certain number of quests in your current zone.

There are no options.


Localization
------------

Compatible with English, Deutsch, Español, Français, Português, Русский,
한국어, 简体中文, and 正體中文 clients.

Translations are not required.
